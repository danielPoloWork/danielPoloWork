## 关于我

我是一名拥有 **10 年以上**专业经验的全栈工程师，专注于企业级 Web 应用（CRM、ERP）和工业自动化的**后端架构**。

过去**三年**，我一直在为**全自动化仓库**设计管理软件 — 通过自定义协议适配器，与**自动导引车（AGV）**和**精密称重硬件**进行实时集成。

最近，我主导了一项 **CAD 自动化**计划，用 **SolidWorks** 宏管线取代了传统的 **2D AutoCAD** 工作流程，直接根据 ERP 数据生成 3D 装配体和 2D 技术图纸。

我最擅长后端（**Java / Spring**），并能胜任全栈工作。我正从高级工程师向架构师级别的后端与平台工作迈进，同时积极探索 **AI 智能体工程**这一新前沿。

> **关于公开内容：** 我最重要的工作大多是受保密协议（NDA）约束的非公开项目。此处公开的内容是为展示同等工程能力而专门构建的参考项目和编程挑战。我很乐意通过通话详细介绍非公开系统的架构与设计决策。

## 当前工程工作

#### 机器人化仓库管理系统
`Java` `Spring Boot` `PostgreSQL` `Docker` `REST`

运行于全自动化仓库内部的企业级 Web 应用。通过自定义协议适配器与 **AGV** 和**精密秤**实时集成；统筹拣选作业、库存核对和运营仪表板。在三年的迭代交付中，我负责后端架构、REST API 设计、数据建模和系统文档。*每天处理约 200 次拣放作业，覆盖约 20,000 个 SKU。*

#### SolidWorks 3D 制图自动化管线
`VBA` `SolidWorks API` `SQL` `PDM` `SMTP`

将机械制图工作流程从 **2D AutoCAD 迁移到 3D SolidWorks** 的宏管线。从公司 ERP 读取结构化数据，处理父文件配置，并自动生成子级 3D 模型及其对应的 2D 图纸。*制图时间缩短约 99%；目前正扩展到完整装配体生成。*

#### 企业级 LLM 知识库 — 本地 AI 记忆 · 研发
`LLM` `RAG` `OCR` `AI Agents`

面向全公司、可由 AI 智能体查询的知识库内部概念验证。将专有文档索引到本地向量库中，使 LLM 智能体无需将数据发送到外部 API 即可获得上下文相关的回答。
